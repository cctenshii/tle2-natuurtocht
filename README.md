# tle2-natuurtocht
Ons concept is om een kaartenverzameling applicatie te maken voor kinderen tussen de 12 en 16 jaar, die vanuit het OERRR-programma komt van Natuurmonumenten. Met onze app verzamel je kaarten die met het natuur te maken heeft, zoals planten, schimmels etc. Per kaart krijg je een aantal kenmerken te zien zodat je daarmee bijvoorbeeld de plant kan vinden in het natuur. Nadat je het hebt gevonden, maak je een foto en word er nagecheckt of het klopt. Als het klopt, krijg je een vraagje over de kaart. Als je de vraag goed beantwoord, word je kaart shiny en als je het fout beantwoord, krijg je hoe dan ook de kaart maar is het geen shiny kaart, maar een normale kaart.
